SQL เป็น Programing language สำหรับจัดการข้อมูล Database เปรียบเเสมือน Excel but coding
โดยที่จัดเก็บข้อมูลใน Relation database คือมี 
1.Primary key, foreign key
2.Table (Rows, Columns)

SQL จำเป็นที่ต้องสร้างตารางขึ้นมาเพื่อเก็บข้อมูลก่อน เมื่อมีข้อมูลตารางนั้นอยู่แล้วจะไม่สามารถรันใหม่ได้ ควรจัดเก็บที่ใส่ข้อมูลอย่างเป็นระเบียบและแยกชื่อไฟล์ในการรันไว้

SQL เวลาเขียนกับเวลาที่คอมพิวเตอร์อ่านจะไม่เหมือนกัน 

SQL จะแยกข้อมูลเป็นหลายตารางและใช้คำสั่ง JOIN มาเจอกัน

----------------------------------------------
การใช้งาน SQL จึงเหมาะกับ
ข้อมูลมีโครงสร้างชัดเจนและไม่ค่อยเปลี่ยนแปลง
เน้นความถูกต้องของข้อมูลสูงมาก (เช่น ระบบธนาคาร, ระบบบัญชี)
ต้องมีการทำ Complex Join หรือความสัมพันธ์ของข้อมูลซับซ้อนมาก
