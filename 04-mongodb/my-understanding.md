MongoDB เป็น noSQL
เป็นการเขียนรูปแบบการเก็บข้อมูลที่อยู่ในรูปแบบคล้ายกับ Javascript
โดยการใช้ BSON ในการสื่อสารระหว่างคอมพิวเตอร์และมนุษย์
มี Pipeline ที่จำแนกข้อมูลได้สะดวกกว่า

เราเริ่มเขียนตารางใส่ข้อมูลไว้ได้เลย ซึ่งสะดวกและเร็วกว่า SQL

เหมาะสำหรับโปรเจ็ค Ecommerce หรือโปรเจ็คที่ต้องการความเร็วในการพัฒนา (แต่ความละเอียดสู้ SQL ไม่ได้)
